# Denis Novik Portfolio

Простий односторінковий вебсайт-портфоліо дизайнера UX/UI Denis Novik.

## Опис

Це статична HTML-сторінка з адаптивним дизайном, створена за допомогою Tailwind CSS та шрифту Open Sans.

## Структура

- `index.html` — головна сторінка портфоліо
- `images/` — папка з фотографіями та SVG-іконками

## Розділи на сторінці

- Home — головна секція
- About me — про автора
- Skills — навички та інструменти
- Portfolio — приклади робіт
- Contacts — контакти та соціальні мережі

## Як запустити

1. Відкрий `index.html` у браузері.
2. Або запусти локальний сервер (наприклад, Live Server у VS Code або будь-який іншій HTTP-сервер).

## Примітки

- Для коректного відображення іконок використовується `images/sprite.svg`.
- Сторінка підходить для перегляду як на десктопах, так і на мобільних пристроях.
